Quick Start

Get up and running with Feyra AI in minutes.

# Install Feyra SDK
pip install feyra-ai
import feyra

client = feyra.Client(api_key="your_key")
response = client.analyze(text="Hello world")

API Endpoints

Endpoint Method Description
/api/v1/analyze POST Analyze text content
/api/v1/vision POST Process images
/api/v1/predict GET Get predictions
